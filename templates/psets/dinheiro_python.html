Dinheiro

Implemente um programa que calcula o número mínimo de moedas necessárias para dar troco a um cliente.

$ python dinheiro.py
Troco devido: 0.41
4

Especificação

  • Escreva, em um arquivo chamado code class="highlighter-rouge">dinheiro.py em um diretório ~/problemas6/dinheiro/, um programa que primeiro pergunta o usuário a quantidade de troco que o cliente deve receber e em seguida informa o número mínimo de moedas com as quais esse troco pode ser dado, exatamente como você fez na Lista de Problemas 1, exceto pelo fato de que seu programa desta vez deve ser escrito em Python.
  • Use a função get_float da biblioteca CS50 para recebe entrada de dados do usuário e a função print para imprimir a resposta. Você deve considerar que apenas moedas de vinte e cinco centavos, dez centavos, cinco centavos, e um centavo estão disponíveis.
    • Nós pedimos que você use a função get_float para que o programa possa lidar com reais e centavos, porém sem que o símbolo cifrão venha antes do valor devido. Em outras palavras, se um cliente deve receber $9.75 de troco, você deve considerar que a entrada de dados para seu programa será 9.75 e não $9.75 ou 975. Entretanto, se o cliente deve receber exatamente $9 de troco, você deve considerar que a entrada de dados para seu programa será 9.00 ou simplesmente 9, mas não $9 ou 900. Evidentemente, considerando a natureza de valores de ponto flutuante, seu programa também funcionará com entradas de dados como 9.0 e 9.000. Você não precisa checar se a entrada de dados de um usuário está "formatada" da forma que dinheiro deve estar.
  • Se o usuário falhar em digitar um valor não negativo, seu programa deve pedir novamente ao usuário por um valor válido até que o usuário colabore.
  • Para que possamos automatizar alguns testes no seu código, nós pedimos que a última linha de saída de dados do seu programa deva ser apenas o número mínimo de moedas possível: um número inteiro seguido de uma nova linha.

Uso

Seu programa deve se comportar como mostrado abaixo.

$ python dinheiro.py
Troco devido: 0.41
4

Testando

Não há code class="highlighter-rouge">check50 para este problema, mas não se esqueça de testar seu código das seguintes maneiras.

  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite 0.41 e aperte Enter. Seu programa deve imprimir 4.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite 0.01 e aperte Enter. Seu programa deve imprimir 1.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite 0.15 e aperte Enter. Seu programa deve imprimir 2.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite 1.60 e aperte Enter. Seu programa deve imprimir 7.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite 23 e aperte Enter. Seu programa deve imprimir 92.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite 4.2 e aperte Enter. Seu programa deve imprimir 18.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite -1 e aperte Enter. Seu programa deve rejeitar esta entrada de dados como sendo inválida, e pedir ao usuário para digitar outro número.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Digite x e aperte Enter. Seu programa deve rejeitar esta entrada de dados como sendo inválida, e pedir ao usuário para digitar outro número.
  • Execute seu programa com python dinheiro.py, e aguarde o programa te pedir por entrada de dados. Não digite nada e aperte Enter. Seu programa deve rejeitar esta entrada de dados como sendo inválida, e pedir ao usuário para digitar outro número.
