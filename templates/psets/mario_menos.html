Mundo 1-1

Próximo ao fim do Mundo 1-1 no jogo Super Mario Brothers, da Nintendo, Mario precisa subir uma pirâmide de blocos alinhados à direita, como mostrado abaixo..

Vamos recriar essa pirâmide usando a linguagem C, porém com texto, usando o símbolo cerquilha, também chamado de jogo-da-velha ou hash (#), como mostrado abaixo. Cada cerquilha possui altura maior do que largura, por isso, a pirâmide em si possui uma altura maior do que sua largura.

       #
      ##
     ###
    ####
   #####
  ######
 #######
########

O programa que nós vamos escrever se chamará mario. E vamos deixar o usuário decidir o quão alta será a pirâmide, pedindo para ele informar um inteiro positivo entre 1 e 8, incluso.

Abaixo está a forma como o programa deve funcionar caso o usuário digite 8 quando for pedido:

$ ./mario
Altura: 8
       #
      ##
     ###
    ####
   #####
  ######
 #######
########

Se o usuário digitar 4, o programa deve funcionar como mostrado abaixo:

$ ./mario
Altura: 4
   #
  ##
 ###
####

Se o usuário digitar 2, o programa deve funcionar como mostrado abaixo:

$ ./mario
Altura: 2
 #
##

E se o usuário digitar 1, o programa deve funcionar como mostrado abaixo:

$ ./mario
Altura: 1
#

Se o usuário não informar um inteiro positivo entre 1 e 8, incluso, o programa deve pedir ao usuário por um inteiro positivo novamente, até que o usuário decida cooperar:

$ ./mario
Altura: -1
Altura: 0
Altura: 42
Altura: 50
Altura: 4
   #
  ##
 ###
####

Como começar? Vamos abordar este problema um passo de cada vez.

Pseudocódigo

Primeiro, vamos criar um novo diretório (uma pasta) chamado mario dentro do diretório problemas1, executando a seguinte linha

~/ $ mkdir ~/problemas1/mario

Adicione um arquivo chamado pseudocodigo.txt dentro do diretório mario.

Escreva dentro de pseudocodigo.txt um pouco de pseudocódigo que implementa este programa, ainda que você não saiba (ainda) como escrever em linguagem C. Não existe uma maneira correta de escrever pseudocódigo, mas frases curtas em português já são o suficiente. Lembre-se de como nós escrevemos o pseudocódigo para encontrar João Silva. É provável que seu pseudocódigo usará (ou implicará o uso de) uma ou mais funções, condições, expressões Booleanas, repetições, e/ou variáveis.

Spoiler

Existe mais de uma maneira de escrever o pseudocódigo, e aqui está uma!

  1. Peça ao usuário para informar a altura
  2. Se altura for menor do que 1 ou maior do que 8 (ou se não for um número inteiro), volte para o passo 1
  3. Repita de 1 até altura:
    1. Na repetição i, imprime um número i de cerquilhas (#) e depois uma nova linha (\n)

Não tem problema alterar seu pseudocódigo depois de ver este aqui, você só não pode copiar e colar este pseudocódigo no seu arquivo!

Pedindo Entrada de Dados

Qualquer que seja seu pseudocódigo, vamos primeiro escrever somente o código em linguagem C que pede ao usuário (e pede novamente, se necessário) por entrada de dados. Crie um novo arquivo chamado mario.c dentro do seu diretório mario.

Agora, modifique o programa mario.c de maneira que ele peça ao usuário para informar a altura da pirâmide, armazenando o dado informado pelo usuário em uma variável, e pedindo novamente caso a entrada de dado do usuário não seja um inteiro positivo entre 1 e 8, incluso. Em seguida, simplesmente imprima o valor dessa variável, dessa forma confirmando (para si mesmo) que você de fato armazenou a entrada de dado do usuário corretamente, como mostrado abaixo.

$ ./mario
Altura: -1
Altura: 0
Altura: 42
Altura: 50
Altura: 4
Armazenado: 4
Dicas
  • Lembre-se que você pode compilar seu programa com make.
  • Lembre-se que você pode imprimir o valor de um int com printf usando %i.
  • Lembre-se que você pode receber um inteiro do usuário com a função get_int.
  • Lembre-se que a funçãoget_int está declarada no arquivo cs50.h.
  • RLembre-se que na aula nós pedimos ao usuário por um inteiro positivo no programa positivo.c.

Construindo o Oposto

Agora que seu programa está recebendo entrada de dados corretamente (assim esperamos), é hora de irmos para o próximo passo.

Acontece que é mais fácil construir uma pirâmide alinhada à esquerda do que à direita, como mostrado abaixo.

#
##
###
####
#####
######
#######
########

Portanto, vamos construir uma pirâmide alinhada à esquerda primeiro e depois, quando essa estiver funcionando, vamos construir uma alinhada à direita.

Modifique o programa mario.c de forma que ao invés de simplesmente imprimir a entrada de dados do usuário, o programa imprima uma pirâmide alinhada à esquerda de altura igual ao número informado pelo usuário.

Dicas
  • Lembre-se que a cerquilha (#) é um caractere como qualquer outro, portanto você pode imprimi-lo usando printf.
  • Assim como Scratch possui o bloco "repita", a linguagem C possui a repetição for, com a qual você pode repetir um bloco de comandos um número de vezes. Talvez, em cada repetição i, você possa imprimir um número i de cerquilhas.

  • Você pode "aninhar" repetições, iterando com uma variável (i, por exemplo) na repetição exterior e com outra variável (j, por exemplo) na repetição interior. Por exemplo, aqui está o algoritmo que você poderia usar para imprimir um quadrado de altura e largura n. Obviamente, não é um quadrado que você precisa imprimir para resolver este problema.

      for (int i = 0; i < n; i++)
  {
      for (int j = 0; j < n; j++)
      {
          printf("#");
      }
      printf("\n");
  }

Alinhando à Direita com Pontos

Vamos alinhar a pirâmida à direita empurrando as cerquilhas para o lado direito imprimindo pontos antes das cerquilhas, como mostrado abaixo. 

.......#
......##
.....###
....####
...#####
..######
.#######
########

Modifique o programa mario.c de maneira que ele faça exatamente isso!

Dica

Perceba que o número de pontos necessários em cada linha é o "oposto" do número de cerquilhas daquela linha. Para uma pirâmide de altura 8, a primeira linha possui apenas 1 cerquilha e 7 pontos. A linha da base, por outro lado, possui 8 cerquilhas e 0 pontos. Usando qual fórmula (aritmética, para sermos precisos) você poderia imprimir a quantidade correta de pontos?

Como Testar Seu Código

Seu programa funciona como prescrito quando você digita

  • -1 (ou outros números negativos)?
  • 0?
  • 1 até 8?
  • 9 ou outros números positivos?
  • letras ou palavras?
  • nenhuma entrada de dado, quando você só aperta Enter?

Removendo os Pontos

Tudo o que resta agora é um acabamento final! Modifique mario.c de tal maneira que ele imprima espaços ao invés de pontos!

Como Testar Seu Código

check50 cs50emportugues/problemas/2020/mario/menos

Execute a linha abaixo para avaliar o estilo do seu código usando a ferramenta style50.

Dica

Um espaço é simplesmente um pressionamento da barra de espaço do seu teclado, assim como um ponto é simplesmente o pressionamento dessa tecla! Lembre-se que printf exige que você envolva ambos caracteres com aspas duplas!
