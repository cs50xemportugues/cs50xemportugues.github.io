Mario

Implemente um programa que imprime metade de uma pirâmide com uma altura específica, como mostrado abaixo.

$ ./mario
Altura: 4
   #
  ##
 ###
####

Especificação

  • Escreva, em um arquivo chamado mario.py em um diretório ~/problemas6/mario/menos, um programa que recria a metade de uma pirâmide usando cerquilhas (#) como sendo os blocos, exatamente como vocês fez na Lista de Problemas 1, exceto pelo fato de que seu programa desta vez deve ser escrito em Python.
  • Para tornar as coisas mais interessantes, primeiro peça ao usuário para informar a altura da meia-pirâmide usando a função get_int, que deve ser um inteiro positivo entre 1 e 8, incluso.
  • Se o usuário não digitar um inteiro positivo que não seja maior que 8, o programa deve pedir ao usuário para informar a altura novamente.
  • Em seguida, o programa deve criar, com a ajuda da função print e de uma ou mais estruturas de repetição, a meia-pirãmide desejada.
  • Tenha atenção para alinhar o canto inferior esquerdo da meia-pirâmide com o lado esquerdo da janela do terminal.

Uso

Seu programa deve se comportar como mostrado abaixo.

$ ./mario
Altura: 4
