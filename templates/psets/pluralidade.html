Pluralidade

Implemente um programa que execute uma eleição em formato pluralidade, como mostrado abaixo.

$ ./pluralidade Ana Bruno Carlos
Número de eleitores: 4
Voto: Ana
Voto: Bruno
Voto: Carlos
Voto: Ana
Ana

Entendendo o Problema

Processos eleitorais existem nas mais variadas formas. No Reino Unido, o Primeiro Ministro é oficialmente escolhido pelo monarca, que normalmente escolhe o lider do partido político que ganhou mais cadeiras na Câmara dos Comuns (o parlamento). Os Estados Unidos usam um sistema de Colégio Eleitoral com várias etapas onde os cidadãos votam em como cada estado alocar "Delegados" que depois elegem o Presidente.

Talvez o jeito mais simples de se fazer uma eleição seja através de um método chamado de "sistema de pluralidade de votos", que por sinal é o tipo de eleição usada no Brasil. No sistema de pluralidade de votos, cada eleitor tem direito de votar em um candidato. No fim da eleição, o candidato que possuir o maior número de votos é declarado o vencedor da eleição.

Entendendo o Problema

Vamos dar uma olhada no arquivo pluralidade.c para entender o código de distribuição que nós providenciamos para você.

A linha #define MAX 9 significa que MAX é uma constante (cujo valor é 9) que pode ser usada no programa. Esta constante representa o número máximo de candidatos que a eleição pode ter.

Em seguida, o arquivo define uma estrutura (struct) chamada candidato. Cada candidato possui dois campos: uma string chamada nome reprensentando o nome do candidato, e um int chamado votos representando o número de votos que o candidato tem. Em seguida, o arquivo define um vetor de candidatos global, onde cada elemento do vetor é um candidato.

Agora, vamos dar uma olhada na função main. Tente encontrar as linhas de código onde o programa define uma variável global chamada numero_candidatos representando o número de candidatos na eleição, copiar os argumentos de linha de comando para o vetor candidatos, e pede ao usuário para digitar o número de eleitores. Em seguida, o programa permite que cada eleitor digite o seu voto (consegue ver como?), chamando a função vvoto para cada candidato que for votado. Por fim, a função main chama a função imprimir_vencedor para imprimir o vencedor (ou vencedores) da eleição.

Entretanto, se você ler o arquivo mais a fundo, você perceberá que as funções voto e imprimir_vencedor estão vazias. É sua tarefa implementá-las!

Especificação

Complete a implementação do programa pluralidade.c de forma que o programa simule uma eleição em formato pluralidade.

  • Complete a função voto
    • a função voto recebe um único argumento, uma string chamada nome, que representa o nome do candidato para o qual o voto foi dado.
    • Se nome for igual ao nome de um dos candidatos na eleição, atualize o número de votos daquele candidato para registrar o novo voto. A função voto neste caso deve retornar true (verdadeiro) para indicar que o voto foi executado com sucesso.
    • Se nome não for igual ao nome de nenhum dos candidatos da eleição, nenhum dos números de votos dos candidatos deve ser alterado, e a função voto deve retornar false (falso) para indicar que o voto é inválido.
    • Você pode considerar que não haverá dois candidados com o mesmo nome.
  • Complete a função imprimir_vencedor
    • A função deve imprimir o nome do candidato que recebeu mais votos na eleição, e em seguida imprimir uma nova linha.
    • É possível que a eleição termine em um empate caso mais de um candidato tenha o número máximo de votos. Nesse caso, você deve imprimir os nomes de todos os candidatos vencedores, cada um em uma linha diferente.

Você não deve modificar mais nada no arquivo pluralidade.c, a não ser as implementações das funções voto e imprimir_vencedor (e a inclusão de arquivos-cabeçalho adicionais, caso você queira).

Uso correto

Seu programa deve se comportar como nos exemplos abaixo.

$ ./pluralidade Ana Bruno
Número de eleitores: 3
Voto: Ana
Voto: Bruno
Voto: Ana
Ana

$ ./pluralidade Ana Bruno
Números de eleitores: 3
Voto: Ana
Voto: Carlos
Voto inválido.
Voto: Ana
Ana

$ ./pluralidade Ana Bruno Carlos
Número de eleitores: 5
Voto: Ana
Voto: Carlos
Voto: Bruno
Voto: Bruno
Voto: Ana
Ana
Bruno
