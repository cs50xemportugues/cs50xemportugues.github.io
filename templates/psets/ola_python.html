Olá

Implemente um programa que diz "olá" para o usuário, como mostrado abaixo.

$ python ola.py
Qual é seu nome?
Ramon
olá, Ramon

Especificação

Escreva, em um arquivo chamado ola.py no diretório ~/problemas6/ola, um programa que pergunta ao usuário pelo seu nome, e em seguida imprime olá, Fulano, onde Fulano é o nome digitado pelo usuário, exatamente como você fez na Lista de Problemas 1, exceto que seu programa desta vez deve ser escrito em Python.

Uso

Seu programa deve se comportar como mostrado abaixo.

$ python ola.py
Qual é seu nome?
David
olá, David

Testando

Não há check50 para este problema, mas não se esqueça de testar seu código das seguintes maneiras.

  • Execute seu programa com python python ola.py, e aguarde o programa te pedir por entrada de dados. Digite David e aperte Enter. Seu programa deve imprimir olá, David.
  • Execute seu programa com python ola.py, e aguarde o programa te pedir por entrada de dados. Digite Ricardo e aperte Enter. Seu programa deve imprimir olá, Ricardo.
