Crédito

Implemente um programa que determina se um número de cartão de crédito informado pelo usuário é válido de acordo com o algoritmo de Luhn.

$ python credito.py
Número: 378282246310005
AMEX

Especificação

  • Escreva, em um arquivo chamado credito.py em um diretório ~/problemas6/credito/, um programa que pede ao usuário para informar um número de cartão de crédito e em seguida informa (através da função print) se o número é um número de cartão de crédito válido da American Express, da MasterCard, ou da Vida, exatamente como você fez na Lista de Problemas 1, exceto pelo fato que seu programa desta vez deve ser escrito em Python.
  • Para que possamos automatizar alguns testes no seu código, nós pedimos que a última linha que seu programa imprimir seja AMEX\n ou MASTERCARD\n ou VISA\n ou INVÁLIDO\n, nada mais, nada menos.
  • Para propósitos de simplicidade, você pode considerar que a entrada de dados do usuário será inteiramente numérica (ou seja, sem hífens, que normalmente são imprimidos juntos dos números em cartões reais).
  • Nós recomendamos que você use a função get_int ou get_string da biblioteca CS50 para receber a entrada de dados do usuário, dependendo de como você decidir implementar este programa.

Uso

Seu programa deve se comportar como mostrado abaixo.

$ python credito.py
Número: 378282246310005
AMEX

Testando

Não há check50 para este problema, mas não se esqueça de testar seu código das seguintes maneiras.

  • Execute seu programa com python credito.py, e aguarde o programa te pedir por entrada de dados. Digite 378282246310005 e aperte Enter. Seu programa deve imprimir AMEX.
  • Execute seu programa com python credito.py, e aguarde o programa te pedir por entrada de dados. Digite 371449635398431 e aperte Enter. Seu programa deve imprimir AMEX.
  • Execute seu programa com python credito.py, e aguarde o programa te pedir por entrada de dados. Digite 5555555555554444 e aperte Enter. Seu programa deve imprimir MASTERCARD.
  • Execute seu programa com python credito.py, e aguarde o programa te pedir por entrada de dados. Digite 5105105105105100 e aperte Enter. Seu programa deve imprimir MASTERCARD.
  • Execute seu programa com python credito.py, e aguarde o programa te pedir por entrada de dados. Digite 4111111111111111 e aperte Enter. Seu programa deve imprimir VISA.
  • Execute seu programa com python credito.py, e aguarde o programa te pedir por entrada de dados. Digite 4012888888881881 e aperte Enter. Seu programa deve imprimir VISA.
  • Execute seu programa com python credito.py, e aguarde o programa te pedir por entrada de dados. Digite 1234567890 e aperte Enter. Seu programa deve imprimir INVÁLIDO.
