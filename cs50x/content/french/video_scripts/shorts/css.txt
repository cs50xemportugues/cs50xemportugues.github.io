Alors, prenons une autre vidéo pour parler d'un autre langage. Cette fois, nous allons parler de CSS. CSS, qui signifie Cascading Style Sheets, est un autre langage que nous utilisons lorsque nous construisons des sites Web. Pensez-y comme ceci. Si HTML est ce que nous utilisons pour organiser le contenu que nous voulons mettre sur notre page, CSS est l'outil que nous utilisons généralement pour personnaliser l'apparence de nos sites Web et l'expérience utilisateur en interagissant avec notre site Web.

Tout comme HTML, CSS n'est pas un langage de programmation. Il n'a pas de logique. Il n'a pas de variables. Il n'a pas de sorte de ces choses liées au flux que C fait. C'est un langage de stylisme. Et sa syntaxe est assez simple et décrit simplement la manière dont les éléments de notre page ont certains éléments HTML qui doivent être modifiés. À cette fin, si vous n'avez pas encore regardé notre vidéo sur HTML ou si vous n'êtes pas familier avec HTML en général, vous voudrez peut-être y jeter un coup d'œil d'abord, car cette discussion de CSS dépendra de certaines connaissances en HTML.

Voici donc une feuille de style CSS très simple. Même si vous n'avez jamais programmé avec CSS auparavant, je suis sûr que vous pouvez comprendre exactement ce que cette feuille de style fait. Que fait-il? Eh bien, appliqué au corps de notre page Web, tout ce qui se trouve entre les balises body de notre HTML, il définit la couleur de fond de cette page en bleu. Eh bien, c'est une feuille de style très simple. CSS est en fait un langage très convivial pour l'utilisateur. Donc, même si vous ne l'avez jamais utilisé auparavant, vous pourriez probablement deviner ce que cela a fait. En fait, si nous chargeons une page dans laquelle cette feuille de style est intégrée de quelque manière que ce soit, la couleur de fond de notre page sera bleue et non blanche.

Comment construisons-nous des feuilles de style? Eh bien, d'abord, nous devons identifier un sélecteur. Dans le dernier exemple, ce sélecteur était body. Ensuite, nous avons une accolade ouverte, et nous allons commencer à définir la feuille de style pour ce sélecteur. Entre les accolades, nous avons simplement une liste de paires clé-valeur. La paire de valeurs précédente était la couleur de fond bleue point-virgule, mais nous pourrions faire plus de ces choses. Vous pouvez avoir plusieurs choses s'appliquant à cette balise, ce sélecteur body. Chacun d'eux est séparé par un point-virgule, et nous appelons chacun d'eux une déclaration, une déclaration CSS.
