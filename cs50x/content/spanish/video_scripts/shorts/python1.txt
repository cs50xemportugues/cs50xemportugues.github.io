A medida que seguimos dejando atrás C, quisiera presentarles otro lenguaje de programación que utilizaremos mucho en CS50: Python. Por supuesto, no podremos profundizar en Python tanto como lo hicimos con C en las 30 o más videos que dedicamos a hablar sobre C en las semanas anteriores, pero el objetivo aquí es darles una introducción al lenguaje para que puedan ver algunas de las herramientas que tiene y descubrir cómo les convendría utilizarlas mejor por su cuenta. Python es un ejemplo de lenguaje de programación moderno bastante común. Probablemente está entre los cinco o seis más usados en el momento en que se está grabando este video. Sin embargo, ha existido por más de 25 años y es una excelente opción de lenguaje para hacer algunas operaciones complejas en C mucho más sencillas. Quizás recuerden que trabajar con manipulación de cadenas de C puede ser un desafío real. También simplifica cosas como la conexión en red y es un lenguaje de utilidad de propósito general que se puede utilizar para hacer muchas cosas. A su vez, es muy popular entre los científicos de datos para procesar grandes conjuntos de datos y generar gráficos, diagramas y resultados a partir de ello.

Hay buenas noticias también. Python está bastante inspirado por C, como de hecho lo están muchos lenguajes de programación modernos. Y aunque la sintaxis de Python será un poco diferente, tiene algunas cosas bastante consistentes en su apariencia y sensación. No verán tantas llaves o algo así que no hayan visto en C, pero espero que algunas de las lecciones de estilo que aprendieron en el camino les sean útiles aquí. Para empezar a escribir un archivo de Python es muy sencillo. Todo lo que hay que hacer es abrir un archivo con la extensión de archivo ".py" dentro de CS50 IDE. Esto automáticamente lo resaltará con sintaxis y le mostrará si lo que está escribiendo es un Python válido o no. Sin embargo, a diferencia de C, Python no es un lenguaje compilado, o al menos no es necesariamente un lenguaje compilado. Los programas de Python pueden ejecutarse en un intérprete de Python. Esto es similar a PHP si están familiarizados con ese idioma también, donde pueden escribir sus líneas de código y hacer que la computadora las ejecute una por una, ejecutándolas a medida que avanzan. Los programas de Python pueden trabajar de la misma manera. 

Un detalle importante antes de sumergirnos en esto. En CS50, enseñamos Python 3. Hay en realidad dos versiones bastante populares de Python, Python 2 y Python 3. Por lo tanto, toda la sintaxis y todo lo que hablaremos en este video es específico de Python 3, y en general, si buscan documentación por su cuenta, tratando de averiguar cómo utilizar una función de Python o averiguar si hay una función de Python que hace algo que están buscando hacer, asegúrense de incluir Python 3 en su búsqueda en lugar de usar solo Python, ya que podrían obtener resultados de Python 2 que no funcionarían necesariamente.

Entonces, vamos a repasar algunas de las cosas básicas que podemos hacer en C y mostrarles cómo podemos hacerlas en Python.
