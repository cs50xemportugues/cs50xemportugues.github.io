Los programadores prefieren el sistema operativo Linux porque es más fresco, ¿verdad?

Muchas distribuciones modernas de Linux tienen interfaces gráficas de usuario, las cuales también llamamos GUI (G-U-I), para permitir una navegación fácil con el mouse, con lo que probablemente esté familiarizado si es usuario de Windows o Mac, moviendo el ratón, haciendo doble clic en los iconos, y así sucesivamente. Sin embargo, como programador, y aunque el IDE contiene la capacidad de hacer algunas cosas de usuario gráfico, hacer clic, arrastrar, y todo eso, aún utilizará con frecuencia su ventana de terminal. Y puede hacer muchas de las mismas tareas que puede hacer con un mouse con comandos de teclado. Vamos a hablar un poco sobre cuáles son algunos de esos comandos ahora mismo.

Estos comandos se pueden utilizar en cualquier sistema operativo basado en Unix, que incluye Linux, pero también incluye macOS. Si abre Terminal en su Mac, puede utilizar estos mismos comandos. Windows también tiene la línea de comandos, pero algunos de los comandos son ligeramente diferentes, por lo que no funciona realmente, porque Windows no es un sistema operativo basado en Unix.

Así que veamos algunos de estos comandos de Linux. El primero que probablemente usará bastante es "ls". Es una "l" minúscula seguida de una "s" minúscula, que significa "listar". Y lo que hace el comando "listar" es darte una lista de todos los archivos y carpetas en tu directorio actual. Así que puedes ver todo a lo que puedes acceder desde donde te encuentras actualmente.
