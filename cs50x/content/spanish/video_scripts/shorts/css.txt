Así que vamos a tomar otro video para hablar de otro lenguaje. Esta vez hablaremos de CSS. CSS, que significa Cascading Style Sheets, es otro lenguaje que usamos al construir sitios web. Piense en ello de esta manera: si HTML es lo que usamos para organizar el contenido que queremos colocar en nuestra página, CSS es la herramienta que generalmente usamos para personalizar cómo se ven nuestros sitios web y cómo es realmente la experiencia del usuario al interactuar con nuestro sitio. 

Similar a HTML, CSS no es un lenguaje de programación. No tiene lógica. No tiene variables. No tiene ningún tipo de ese flujo de cosas relacionadas con C. Es un lenguaje de estilo. Y su sintaxis es bastante simple, y simplemente describe cómo se deben modificar los elementos de nuestra página con ciertos elementos HTML. Para ese fin, si aún no ha visto nuestro video sobre HTML o no está familiarizado con HTML en general, es posible que desee echar un vistazo primero, porque esta discusión de CSS dependerá de cierto conocimiento de HTML. 

Entonces aquí hay una hoja de estilo CSS muy simple. Incluso si nunca ha programado con CSS antes, estoy bastante seguro de que puede averiguar exactamente qué hace esta hoja de estilo. ¿Qué hace? Bueno, aplicado al cuerpo de nuestra página web, todo entre las etiquetas del cuerpo en nuestro HTML, y establece el color de fondo de esa página en azul. Bueno, es una hoja de estilo muy simple. En realidad, el lenguaje CSS es muy amigable para los humanos. Por lo tanto, incluso si nunca lo ha usado antes, probablemente podría adivinar lo que hizo. De hecho, si cargáramos una página donde esta hoja de estilo se incorporara de alguna manera, el color de fondo de nuestra página sería azul, y no el blanco estándar. 

Entonces, ¿cómo construimos hojas de estilo? Bueno, primero tenemos que identificar un selector. En el último ejemplo, ese selector fue cuerpo. Luego tenemos una llave abierta, y comenzamos a definir la hoja de estilo para ese selector. Entre las llaves rizadas, tenemos una lista de pares clave-valor. El par de valores previo fue color de fondo azul punto y coma, pero podríamos hacer más y más de estos. Podría haber múltiples cosas aplicándose a esa etiqueta, ese selector de cuerpo. Cada uno de ellos está separado por un punto y coma, y llamamos a cada uno de ellos una declaración, una declaración CSS. Cuando terminamos con todo el estilo que queremos aplicar a esa etiqueta en particular, simplemente tenemos una llave rizada de cierre para finalizar la hoja de estilo, y hemos terminado de definir la hoja de estilo para ese selector en particular.
