Vamos fazer mais um vídeo para falar sobre mais uma linguagem. Dessa vez, vamos falar sobre CSS. CSS, que é a abreviação para Cascading Style Sheets, é outra linguagem que usamos ao construir sites. Pense nisso assim: se o HTML é o que usamos para organizar o conteúdo que queremos colocar em nossa página, o CSS é a ferramenta que geralmente usamos para personalizar como nossos sites parecem e qual é a experiência do usuário ao interagir com nosso site.

Assim como o HTML, o CSS não é uma linguagem de programação. Ele não tem lógica. Não tem variáveis. Não tem nenhum tipo de fluxo relacionado como em C. É uma linguagem de estilo. E sua sintaxe é bastante simples e descreve apenas como os elementos de nossa página têm determinados elementos HTML que devem ser modificados. Para tanto, se você ainda não assistiu a nosso vídeo sobre HTML ou não está familiarizado com o HTML em geral, convém dar uma olhada primeiro, porque esta discussão sobre CSS dependerá de algum conhecimento de HTML.

Aqui está uma CSS simples. Mesmo que você nunca tenha programado com CSS antes, estou bastante certo de que você pode entender exatamente o que esta folha de estilo faz. O que faz? Bem, aplicado ao corpo de nossa página da web, tudo entre as tags do corpo em nosso HTML e define a cor de fundo daquela página para azul. Bem, é uma folha de estilo muito simples. Na verdade, o CSS é uma linguagem muito amigável para humanos. Então, mesmo se você nunca o usou antes, provavelmente poderia adivinhar o que isso fez. Na verdade, se carregássemos uma página onde essa folha de estilo estivesse incorporada de alguma forma, a cor de fundo de nossa página seria azul e não branca, como de costume.

Então, como construímos folhas de estilo? Bem, primeiro, temos que identificar um seletor. No último exemplo, esse seletor era o corpo. Então, temos uma chave de abertura e começamos a definir a folha de estilo para esse seletor. Entre as chaves, temos apenas uma lista de pares chave-valor. O par anterior era cor de fundo azul ponto e vírgula, mas poderíamos fazer isso com mais e mais desses. Você poderia ter várias coisas aplicando a esse tag, o seletor do corpo. Cada um deles é separado por um ponto e vírgula, e chamamos cada um deles de declaração, uma declaração CSS. Quando terminamos de estilizar tudo o que queremos aplicar a essa tag específica, temos apenas uma chave de fechamento para encerrar a folha de estilo, e terminamos de definir para aquele seletor específico.
