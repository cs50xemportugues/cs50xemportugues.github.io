Scratch

É hora de embarcar na sua própria aventura! Seu dever é implementar um projeto da sua preferência usando Scratch! Esse projeto pode ser uma história iterativa, um jogo, uma animaçao, ou qualquer outra coisa, desde que siga os requisitos abaixo:

  • Seu projeto precisa ter pelo menos dois atores (personagens), dentre os quais um dos atores precisa ser algo que não seja um gato.
  • Seu projeto precisa ter pelo menos três scripts ("subprogramas"), não necessariamente três por personagem.
  • Seu programa precisa ter pelo menos uma estrutura condicional, uma estrutura de repetição, e uma variável.
  • Seu projeto precisa usar pelo menos um arquivo de áudio.
  • Seu projeto deve ser mais complexo do que a maioria dos exemplos mostrados na aula (muitos dos quais, ainda que instrutivos, eram bem curtos) mas o projeto pode ser menos complexo do que O Jogo Mais Difícil de Ivy. Além de seguir esses requisitos, é interessante que seu projeto utilize algumas dezenas de blocos/peças.

Se você quiser testar alguns projetos Scratch de ex-alunos, aqui estão alguns:

  • Ivy’s Hardest Game, from lecture
  • Soccer, a game
  • Cookie Love Story, an animation
  • Gingerbread tales, an interactive story
  • Intersection, a game

Você pode usar estes tutoriais ou projetos iniciantes para começar. E nós encorajamos você a explorar o scratch.mit.edu para encontrar inspirações. Mas tente pensar em uma ideia própria, para depois começar a implementá-la. De qualquer maneira, não tente implementar seu projeto completo de uma só vez: dê um passo de cada vez. Em outras palavras, implemente o projeto de pouco a pouco: escreva um pouco de código (arraste e solte algumas peças), teste, escreva mais um pouco, teste, e assim por diante. E selecione a opção Arquivo > Salvar agora a cada nova funcionalidade implementada para que você não perca nenhum progresso!

Se ao longo da jornada, você achar muito difícil implementar alguma funcionalidade, tente não se estressar; altere o design do projeto ou tente contornar o problema. Se você decidir implementar uma ideia que você ache divertida, é bem provável que você não achará tão difícil satisfazer os pré-requisitos acima.

Perfeito! Você está pronto para começar! Dê o seu melhor!

Quando você terminar o seu projeto, selecione Arquivo > Salvar agora uma última vez. Em seguida, selecione Arquivo > Baixar para o seu computador e guarde esse arquivo para que você possa nos enviá-lo.

Olá, Mundo

Nós sabemos que é difícil encontrar colegas de turma quando se está fazendo um curso online. Mas, graças à tecnologia, todo mundo pode dizer pelo menos um "olá".

Se você possui um telefone (ou uma câmera digital) e gostaria de dizer "olá" para seus colegas de turma, grave um vídeo de si próprio, de 1 a 2 minutos de vídeo, dizendo "olá" e também dizendo o lugar no mundo onde você se encontra, por que você está fazendo o CS50x, e mais algo de interesse seu! Tente começar o seu vídeo dizendo "olá, mundo" e finalizá-lo com "meu nome é ..., e este é o CS50." Mas, de qualquer maneira, você é quem decide o que dizer.

Se você gravar esse vídeo, envie ele para o YouTube para que depois você possa nos enviar o URL do vídeo quando você enviar a solução deste problema!

Como Enviar

Passo 1 de 2

Envie este formulário.

Passo 2 de 2

Este passo exige que você tenha baixado seu projeto Scratch como sendo um arquivo cujo nome termina em .sb3. E este passo também exige que você tenha criado uma conta no GitHub, como pedido no formulário acima.

  1. Visite este link, entre na sua conta do GitHub, e clique em Authorize cs50.
  2. Marque a opção "I understand this will..." para que você nos dê permissão para ter acesso aos seus envios, e depois clique em Join course para participar do curso.
  3. Vá ao site https://submit.cs50.io/upload/cs50/problems/2020/x/scratch.
  4. Clique na opção "Choose File" para escolher um arquivo. Em seguida, selecione seu arquivo com extensão .sb3. Por fim, clique em Submit para enviar seu projeto.

É isso! Quando seu envio for confirmado, você deverá ser redirecionado para sua página de envios. Clique no link do envio e, em seguida, clique no link do check50 para ver quais requisitos seu projeto cumpriu. Você pode reenviar seu projeto quantas vezes quiser!

Para ver seu progresso atual no curso, visite o site cs50.me/cs50x!
